Guía Rápida de Bootstrap 4

Resumen de los componentes y clases más importantes de Bootstrap 4.

Botones (`btn`)

Los botones usan la clase btn, y los estilos se cambian con variantes como: btn-primary, btn-secondary, btn-success, entre otros.

Tablas (`table`)

Usa la clase table para tablas simples. Agrega table-striped para líneas alternadas, table-hover para resaltar filas al pasar el cursor, y table-bordered para bordes.

Formularios (`form`)

Los formularios se crean con la clase form-group para agrupar campos y form-control para los inputs.

Tarjetas (`card`)

Las tarjetas usan card, con variantes como card-body, card-title, y card-text.

Grid System (`grid`)

El sistema de rejilla de Bootstrap permite organizar el contenido en columnas. Se usa dentro de un contenedor, y cada fila se divide en 12 columnas. Utiliza las clases col, col-* para personalizar el tamaño de las columnas en distintos tamaños de pantalla.

Offset (`offset-`) y `d-flex`

Las clases offset-* permiten mover columnas hacia la derecha, agregando un espacio vacío antes de la columna. Esto es útil para controlar la alineación sin necesidad de agregar márgenes manualmente.

Medidas Responsivas

Bootstrap tiene clases responsivas que permiten cambiar la visualización en función del tamaño de la pantalla. Estas clases incluyen: col-sm-*, col-md-*, col-lg-*, col-xl-*.

Cuadrículas Responsivas con Bootstrap 4

Bootstrap 4 usa un sistema de cuadrícula de 12 columnas. Puedes dividir estas columnas para adaptarse a diferentes tamaños de pantalla como:

  • col-: Para dispositivos pequeños.
  • col-sm-: Para pantallas mayores a 576px.
  • col-md-: Para pantallas mayores a 768px.
  • col-lg-: Para pantallas mayores a 992px.
  • col-xl-: Para pantallas mayores a 1200px.
